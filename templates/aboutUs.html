ABOUT US

MISSION

We aimed to use data from Twitter to determine the most canceled users on the platform based on trending tweets. As an extra, we included a page with random tweets from all over the world.

TECHNOLOGY USED

We used web scraping to gather data, and stored it in a SQL database. This data was analyzed using sentiment analysis. Flask sent data to the frontend, created by HTML/CSS and a Bootstrap framework.

WHO ARE WE

This project was created by a group of college students during a 36 hour hackathon. We are all very sleep deprived but managed to get the program up and running. There was free food.
